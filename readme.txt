Phil's Bad to the Bone Adventure

Description:

This is a simple javascript app I made to learn more about javascript, as well as make something silly.

Simple sidescroller where Phil escapes after being bad to the bone.
Click Start Button to Start Game.  Use left click to jump over obstacles and beat your highscore.

Technologies:

I mainly utilized javascript and css.  Node installation was later added to include scss.

How to Play:

To play simply download the .zip file and run the index.html.

Features to be added:  Restart button, saved highscore, different obstacle model, explosion animation on death, and tweaking hit detection to be slightly more accurate.
